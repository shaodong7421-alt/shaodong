理论依据与演示算法说明 从传播理论到课堂模拟公式

核心指标理论依据

舆情热度

理论依据：议程设置理论、信息扩散理论、舆情生命周期理论。

课堂解释：事件被看见、讨论和转发的强度，体现公众注意力集中程度。

算法含义：由负面传播节点、关键节点放大和平台扩散速度共同推高，处置后随可信信息覆盖下降。

负面情绪占比

理论依据：情绪传播理论、风险感知理论、群体极化理论。

课堂解释：食品安全议题天然牵动健康风险，公众容易先产生愤怒、恐慌或质疑。

算法含义：受爆料可信度、谣言强度和回应延迟影响，第三方检测与监管回应可降低该比例。

谣言再传播率

理论依据：谣言传播理论、SIR/SEIR 类扩散模型。

课堂解释：未经证实的信息是否继续感染新的传播节点。

算法含义：对应传播概率，受到意见领袖放大影响，也受到平台治理与澄清触达抑制。

官方信息触达率

理论依据：危机传播理论、信源可信度理论、两级传播理论。

课堂解释：权威回应不仅要发布，还要到达用户、被理解并进入讨论链路。

算法含义：由官方回应速度、监管节点、媒体节点和第三方检测可信度共同提升。

澄清转化率 / 谣言节点回落比例

理论依据：说服传播理论、框架理论、纠偏传播研究。

课堂解释：衡量原本扩散负面信息的用户是否停止转发、转为观望或接受澄清。

算法含义：当可信信息触达率高于谣言再传播率时，负面节点逐步转为澄清或降温状态。

处置综合评分

理论依据：危机管理效果评估框架。

课堂解释：综合评估回应是否及时、信息是否可信、热度是否回落、情绪是否缓和。

算法含义：按热度下降、触达率、负面情绪降低和谣言回落加权计算，用于课堂比较不同处置策略。

教学演示算法说明

本页面的演示算法是“教学用加权扩散模型 + S 型增长/衰减函数”，用于解释舆情传播机制和处置反馈，不是现实平台的预测模型。所有滑块参数先归一化到 0-1：C 为初始爆料可信度，R 为谣言传播强度，L 为意见领袖放大倍数，G 为官方回应速度，P 为平台治理力度，T 为第三方检测报告可信度。

B = 0.35C + 0.35R + 0.30L 风险扩散强度。爆料越可信、谣言越强、意见领袖越活跃，事件越容易扩散和升温。
I = 0.38G + 0.30P + 0.32T 处置干预强度。官方回应、平台治理和第三方检测共同构成压低舆情风险的治理合力。
S(x) = 1 / (1 + e^(-x)) S 型函数模拟舆情“慢启动、快速爆发、逐步稳定”的生命周期变化。
H(t) = 18 + 95B·Growth(t)·PhaseBoost - 62I·Decay(t)·Action(t) 舆情热度。扩散因素推高热度，处置因素在介入后促使热度回落。
N(t) = 18 + 76B·Growth(t) - 50I·Action(t) + 8C 负面情绪占比。食品安全风险感知会推高负面情绪，可信处置会降低情绪压力。
M(t) = 8 + 74R·Growth(t) + 22L - 58P·Action(t) 谣言再传播率。谣言强度和关键节点会放大再传播，平台治理会抑制重复谣言与误导内容。
A(t) = 6 + 82I·Action(t) + 4·Phase 官方信息触达率。回应越快、协同越强、检测越可信，权威信息越容易进入讨论链路。
Score = 28 + 0.24D + 0.28A(t) + 0.22Q + 0.18(100 - N(t)) 处置综合评分。综合热度下降率 D、官方触达、谣言回落 Q 和负面情绪缓和程度，用于课堂比较不同策略。

舆情预测典型模型 释义

真实模型思想的简化教学交互
一般结构：真实舆情预测模型通常可概括为 Y(t+k)=Model(Xt)：其中 Xt 是当前时刻以前收集到的传播规模、传播速度、情绪倾向、话题语义、用户网络、媒体介入、官方回应和历史相似案例等特征；Y(t+k) 是未来 k 个时间单位后的预测结果，例如未来热度、负面情绪比例、风险等级、升级概率或传播路径。
